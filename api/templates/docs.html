API Endpoints

Available Endpoints

GET /artikel

Retrieves a list of articles from Google Spreadsheet.

Query Parameters:
  • sheet_id (required)
  • gid (required)
  • page (optional, default = 1)
  • limit (optional, default = 10)
Example Response: 
{
  "payload": {
    "pagination": {
      "halaman_sekarang": 1,
      "total_artikel": 100,
      "total_halaman": 10
    },
    "artikel": [
      {
        "title": "Article Title",
        "category": "Category",
        "date": "2024-03-20",
        "slug": "article-slug"
      }
    ]
  },
  "status": "success",
  "message": "Data berhasil diambil"
}

GET /artikel/detail

Retrieves a single article by its slug.

Query Parameters:
  • slug (required)
Request Body: 
{
  "sheet_id": "...",
  "gid": "..."
}
Example Response: 
{
  "payload": {
    "title": "Article Title",
    "content": "Article content...",
    "category": "Category",
    "date": "2024-03-20",
    "slug": "article-slug"
  },
  "status": "success",
  "message": "Artikel ditemukan"
}

GET /artikel/category

Retrieves articles by category.

Query Parameters:
  • sheet_id (required)
  • gid (required)
  • category (required)
  • page (optional, default = 1)
  • limit (optional, default = 10)
Example Response: 
{
  "payload": {
    "pagination": {
      "halaman_sekarang": 1,
      "total_artikel": 25,
      "total_halaman": 3
    },
    "artikel": [
      {
        "title": "Article Title",
        "category": "Category",
        "date": "2024-03-20",
        "slug": "article-slug"
      }
    ]
  },
  "status": "success",
  "message": "Artikel dengan kategori 'Category' berhasil diambil"
}

GET /artikel/category-terbanyak

Retrieves categories with the most articles.

Query Parameters:
  • sheet_id (required)
  • gid (required)
  • top (optional, default = 10)
Example Response: 
{
  "payload": [
    {
      "category": "Politik",
      "count": 25
    },
    {
      "category": "Ekonomi",
      "count": 20
    },
    {
      "category": "Teknologi",
      "count": 15
    }
  ],
  "status": "success",
  "message": "Kategori terbanyak berhasil diambil"
}
